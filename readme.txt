== Description ==

Allow to select city and display widget with WordPress 10th Anniversary.

= Translators =

* Polish (pl_PL) - [Marcin Pietrzak](http://iworks.pl/)

If you have created your own language pack, or have an update of an existing one, you can send [gettext PO and MO files](http://codex.wordpress.org/Translating_WordPress) to me so that I can bundle it into WP Anniversary. You can [download the latest POT file from here](http://plugins.svn.wordpress.org/wp-anniversary/trunk/languages/upprev.pot).

== Installation ==

1. Upload WP Anniversary to your plugins directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure WP Anniversary plugin using Settings -> WP Anniversary

== Frequently Asked Questions ==

= My website is not in English, will WP Anniversary work? =

WP Anniversary plugin will work on websites in the following languages:
